# 开始使用 {#title}

## 安装 {#install}

<ApkTable />

或前往 [Github Releases](https://github.com/gkd-kit/gkd/releases/latest) 下载

下载完毕后, 在您的 Android 手机上安装并打开

## 使用 {#usage}

打开后进入 GKD 的首页, 要想 GKD 正常工作需要授权无障碍权限

点击无障碍权限右侧的 `授权` 按钮即可前往设置的无障碍界面授权
